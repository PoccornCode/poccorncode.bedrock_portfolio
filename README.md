## Main Project: Fungus Hazard

Fungus Hazard is a supplement designed to prevent the spread of fungal infections.

### Features
- Global infection progression system
- Entities and blocks contribute to evolution phases
- State-driven entity behavior
- Animation controllers integrated with behavior
- Performance-aware spreading system

### Technical Details:
- JavaScript (Scripting API)
- JSON (entities, behaviors, components)
- Custom block states and conditions

### Key Systems

- Infection System
A scalable system in which the infection spreads based on the actions of the player and the infected.

- Evolution System
A global point-based system that modifies the behavior of entities and the difficulty over time; between global evolutions, custom effects are applied that boost the infected’s stats.

- Entity Behavior
Entities react dynamically to the player’s actions and the environment, including adaptive mechanics.
